# J-MDMの導入で分断されていたマスタデータの一元管理を実現データの見える化で社員のデータ活用意識も向上


ロート製薬株式会社では、各業務システムごとに、マスタデータを個別管理していたため、システム間での連携が難航していました。さらに、部署ごとに異なる略称やコードの運用により、社員間、部門間の認識にズレが生まれ、業務上の情報共有や確認作業の負担が大きくなっていました。そこで、マスタデータの一元管理として、株式会社JSOLが提供するJ-MDMを導入。これまで分断されていた各業務システムのマスタデータが、J-MDMに集約されたことで部門間の情報共有がスムーズになり、また、各データを確認できるようになったことで、全社員が単なる入力・参照にとどまらず、データの活用目的や業務への貢献を意識する機会が増えました。本事例の詳細は、下記をご参照ください。https://www.jsol.co.jp/casestudy/094_rohto.html※外部リンクに飛びます

※J-MDMとは株式会社JSOLが提供しているintra-martをベースとしたマスタ統合管理基盤です
