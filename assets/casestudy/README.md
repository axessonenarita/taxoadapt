# 会社情報の入力方法

## 概要

Markdownファイルに会社情報（業種・売上規模）を埋め込むことで、自動的に`company_info.json`が生成されます。以下の2つの方法で入力できます。

## 方法1: YAMLフロントマター（推奨）

Markdownファイルの**先頭**にYAMLフロントマター形式でメタデータを追加します。

### 入力形式

```markdown
---
company_name: 東急不動産ホールディングス株式会社
company_industry: 不動産
company_revenue_size: 1000億円以上
url: https://www.intra-mart.jp/case-study/006468.html
---

# タイトル
...
```

### フィールドの説明

- **`company_name`**: 会社名（自動抽出される会社名と一致させることを推奨）
- **`company_industry`**: 業種（下記の例から選択）
- **`company_revenue_size`**: 売上規模（下記の4つのカテゴリから選択）
- **`url`**: 事例のURL（オプション）

### 代替フィールド名

以下のフィールド名も使用可能です（互換性のため）：

- `industry` → `company_industry` の代替
- `revenue_size` → `company_revenue_size` の代替

## 方法2: 基本情報セクション内

「基本情報」セクション内の会社名の下に、業種と売上規模を追加します。

### 入力形式

```markdown
## 基本情報

### 東急不動産ホールディングス株式会社

業種: 不動産
売上規模: 1000億円以上

所在地
...
```

### フィールドの説明

- **業種**: `業種: 不動産` のように記入（コロンは全角・半角どちらでも可）
- **売上規模**: `売上規模: 1000億円以上` のように記入

## 業種の例

以下の業種から選択してください：

- 不動産
- 製造業
- 金融
- 小売
- IT
- 建設
- 運輸
- 卸売
- サービス
- 医療
- 教育
- 食品
- 化学
- 機械
- 電気
- 通信
- エネルギー
- 鉄鋼
- 非鉄金属
- 繊維
- パルプ
- ガラス
- セラミックス
- 商社
- 保険
- 証券
- 銀行

## 売上規模の指定

以下の4つのカテゴリから選択してください：

- `1000億円以上`
- `500億～1000億円`
- `100億～500億円`
- `100億円未満`

## 優先順位

情報の読み込みは以下の優先順位で行われます：

1. **YAMLフロントマター**（最優先）
2. 基本情報セクション内の「業種:」「売上規模:」形式
3. 手動入力ファイル（`datasets/casestudy/company_info.json`）

## 入力例

### YAMLフロントマター形式

```markdown
---
company_name: 東急不動産ホールディングス株式会社
company_industry: 不動産
company_revenue_size: 1000億円以上
url: https://www.intra-mart.jp/case-study/006468.html
---

# 業務プロセス改革の成果を実装した新たな決裁システムintra-martで実現したシステム全域の最適化とデータ活用の基盤づくり

本文...
```

### 基本情報セクション形式

```markdown
## 基本情報

### 東急不動産ホールディングス株式会社

業種: 不動産
売上規模: 1000億円以上

所在地

東京都渋谷区道玄坂1-21-1 渋谷ソラスタ

設立

2013年10月

事業内容

都市開発・戦略投資・管理運営・不動産流通事業を行う東急不動産ホールディングスグループのグループ経営管理を担う
```

## 注意事項

- YAMLフロントマターはファイルの**先頭**に配置してください
- YAMLフロントマターは `---` で囲む必要があります
- フィールド名と値の間は `:` で区切ります
- 値にクォート（`"` や `'`）は不要ですが、使用しても問題ありません
- 基本情報セクション形式では、「業種:」と「売上規模:」の形式で記入してください（コロンは全角・半角どちらでも可）
- 実行時に自動的に`datasets/casestudy/company_info.json`が生成・更新されます
