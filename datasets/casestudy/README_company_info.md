# 会社情報入力ファイルの使い方

## ファイル名
`datasets/casestudy/company_info.json`

## 入力形式

JSON形式で、会社名をキーとして、業種と売上規模を指定します。

```json
{
  "会社名1": {
    "industry": "業種",
    "revenue_size": "売上規模"
  },
  "会社名2": {
    "industry": "業種",
    "revenue_size": "売上規模"
  }
}
```

## 業種の例
- 不動産
- 製造業
- 金融
- 小売
- IT
- 建設
- 運輸
- 卸売
- サービス
- 医療
- 教育
- 食品
- 化学
- 機械
- 電気
- 通信
- エネルギー
- 鉄鋼
- 非鉄金属
- 繊維
- パルプ
- ガラス
- セラミックス
- 商社
- 保険
- 証券
- 銀行

## 売上規模の指定
以下の4つのカテゴリから選択してください：

- `"1000億円以上"`
- `"500億～1000億円"`
- `"100億～500億円"`
- `"100億円未満"`

## 入力例

```json
{
  "東急不動産ホールディングス株式会社": {
    "industry": "不動産",
    "revenue_size": "1000億円以上"
  },
  "株式会社日立ソリューションズ": {
    "industry": "IT",
    "revenue_size": "1000億円以上"
  },
  "トヨタ自動車株式会社": {
    "industry": "製造業",
    "revenue_size": "1000億円以上"
  },
  "イオン株式会社": {
    "industry": "小売",
    "revenue_size": "1000億円以上"
  }
}
```

## 注意事項

- 会社名は、Markdownファイルから自動抽出される会社名と一致させる必要があります
- 「株式会社」「有限会社」などの表記は、Markdownファイル内の表記に合わせてください
- 部分一致でも検索されますが、完全一致を推奨します
- ファイルが存在しない場合や、会社名が見つからない場合は、業種と規模は`null`になります
